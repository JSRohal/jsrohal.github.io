// PROJECTS

> job_application_tracker

Self-hosted Node.js/Express dashboard that normalizes job-application statuses into a fixed funnel and computes response, interview, and offer rates.

> guardian_state_recognizer

End-to-end AWS ML pipeline for real-time driver-state recognition using YOLOv8. CDK-deployed Lambda streams gzipped S3 uploads and writes SageMaker manifests.

> cancer_predictor

ID3 decision tree built from scratch in Python classifying cancer vs. non-cancer gene-mutation samples, with a Cython-accelerated entropy hot path compiled at -O3/-march=native.

> cgra_hw_security_research

Research into CGRA security using CGRA-Flow, VectorCGRA, Garnet, and PyMTL3. Investigated a published CVE and implemented a CGRA-based cache timing-attack countermeasure.

> systemverilog_cpu

Single-cycle MIPS-style CPU in SystemVerilog with full datapath: PC, register file, ALU, sign-extension, and branch/jump logic. End-to-end verified with Verilator.

> parallel_hash_cracker

Distributed SHA-256 password cracker in C with OpenMPI. Partitions the search space across ranks; early-termination messaging short-circuits the cluster on first match.

> custom_bash_shell

Unix shell in C++ using Flex and Bison. Supports argument parsing, I/O redirection including append for stdout/stderr, and full pipelines via fork/exec.

> wordle_clone

Full-stack Wordle clone built collaboratively with React, Node, Tailwind, and Supabase. Ships a solver algorithm, Jest unit tests, UML diagrams, and bonus minigames.

> spotify_karaoke_app

Full-stack karaoke app with Spotify OAuth 2.0, real-time playback control, and synchronized lyrics via Lyric Finder API. React/Router front end with Node.js/Express back end.

> arduino_rtc_led_sign

Standalone Arduino clock and scrolling message display. An RTC module keeps accurate time across power cycles; C/C++ firmware drives the LED matrix via I2C with managed refresh.
